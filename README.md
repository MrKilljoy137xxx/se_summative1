# Office Attendance Tracker

## 1. Introduction
**Office Attendance Tracker** is a web-based program intended to optimise staff attendance monitoring.  This enables employees to compute their attendance rates by comparing their office presence to their total working days.

This product corresponds with my governmental organisation's overarching objective of improving workplace efficiency via digital transformation. Through the automation of attendance monitoring, we intend to minimise manual errors, conserve time, and promote a more transparent workplace.

This documentation caters to both technical and non-technical readers.  It encompasses user instructions, developer setup manuals, feature summaries, and project planning insights to facilitate continuous development and utilisation.
